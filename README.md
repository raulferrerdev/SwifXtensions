# SwifXtensions

Some useful extensions for Swift.

## UIView+Extension

### Get view dimensions

**width**: CGFloat  
**height**: CGFloat   
**size**: CGSize  
**origin**: CGPoint  
**x**: CGFloat    
**y**: CGFloat  
**centerX**: CGFloat   
**centerY**: CGFloat    
**left**: CGFloat  
**right**: CGFloat  
**top**: CGFloat  
**bottom**: CGFloat   

### Round view corners
**round(corners: UIRectCorner, radius: CGFloat)**

### Get parent view controller
**parentViewController**
